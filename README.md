# Sea Design

🌊 An enterprise-class UI design language and React UI library.

## ⌨️ Development

```bash
$ git clone git@github.com:sea-design/sea-design.git
$ cd sea-design
$ yarn
$ yarn start
```

## 📖 Doc Generation

```bash
$ yarn docs
```

Commit `docs` folder and view site https://sea-design.github.io/sea-design/
